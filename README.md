# halobios_popularization
# 海洋生物科普Web App

#### 介绍
向广大群体科普海洋生物知识，为想要了解海洋生物的人群，提供一个线上科普平台

#### 技术栈
##### 前端
- ###### 前端开发工具（前端2.0文件夹）：Vue2.0
-  前台组件库：Vant
-  后台组件库：Element UI
-  前端构建工具：Webpack
- ###### 前端开发工具（前端3.0文件夹）：Vue3.0
-  前台组件库：Vant
-  后台组件库：Element UI
-  前端构建工具：Vite
-  其他：TS+Less
##### 后端：Spring Boot
- 数据库：MySQL
